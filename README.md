# Cars — лендинг аренды премиальных автомобилей

Современный одностраничный сайт для сервиса аренды автомобилей: каталог, фильтрация по маркам, блок преимуществ и форма заявки.

## Что это за проект

`Cars` — production-ready фронтенд-лендинг, подготовленный для демонстрации заказчику:

- аккуратная визуальная подача и единый бренд-стиль;
- фокус на конверсию: CTA-кнопки, каталог, понятный блок заказа;
- оптимизированный сборочный вывод (минифицированные JS/CSS-ассеты);
- готовность к статическому хостингу (GitHub Pages, Netlify, Vercel Static).

## Ключевые возможности

- Каталог автомобилей с визуальными карточками.
- Фильтрация по брендам.
- Форма заказа с валидацией полей.
- Адаптивное и плавное UX-поведение (скролл, якоря, акценты на действиях).

## Технологии

- HTML5
- CSS3
- JavaScript (сборочный бандл)
- Статическая поставка ассетов (изображения + минифицированные скрипты)

## Быстрый запуск

### Вариант 1: Python

```bash
python3 -m http.server 8080
```

Откройте: `http://localhost:8080`

### Вариант 2: Node.js (serve)

```bash
npx serve . -l 8080
```

Откройте: `http://localhost:8080`

## Деплой

Проект можно публиковать как статический сайт без серверной части:

- **GitHub Pages**
- **Netlify**
- **Vercel (Static Site)**

> Если деплой выполняется в подпапку (например, `/cars/`), убедитесь, что значение `base href` в `index.html` соответствует целевому пути.

## Рекомендации для production

- Подключить веб-аналитику (GA4 / Яндекс Метрика).
- Добавить политику обработки персональных данных рядом с формой.
- Настроить отправку лидов в CRM/почту через API или serverless endpoint.
- Добавить favicon и social preview-изображение для шаринга в соцсетях.
